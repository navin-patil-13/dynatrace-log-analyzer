Log Analyzer

Supported formats: JSON, CSV

Upload your log files for analysis.

After uploading, you can use the API endpoints to search and analyze your logs.

Example API endpoints:

  • GET /api/logs/ - Get all logs
  • GET /api/logs/search?host=example.com&level=ERROR - Search logs by host and level
  • GET /api/logs/server?host=example.com - Search logs by host
